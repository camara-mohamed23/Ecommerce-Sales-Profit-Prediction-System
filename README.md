# EcomInsightAI

AI-powered E-commerce Analytics & Profit Prediction System.

## Features

- Data Cleaning
- Feature Engineering
- Profit Prediction using Random Forest
- Model Evaluation
- Streamlit Dashboard

## Tech Stack

- Python
- Pandas
- Scikit-Learn
- Streamlit

## How to Run

1. Install dependencies:
   pip install -r requirements.txt

2. Train model:
   python main.py

3. Launch dashboard:
   streamlit run app/streamlit_app.py
